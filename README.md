## Local Development

```bash
cd D:\conference\learn-guide
npm install
npm run dev       # http://localhost:4321
```

## Deploy

```bash
npm run build
vercel --prod --yes
```

## Design System

- Dark edu-tech theme (Algorithmics-inspired)
- Animated floating orbs + CSS grid overlay
- Colors: `--accent: #ff6b9d` · `--accent-secondary: #4af0e8` · `--violet: #a855f7`
- Buttons: `.btn.btn-primary` (pink→purple gradient) · `.btn.btn-secondary` (glassmorphism)
- Note: Layout.astro uses `<style is:global>` so `.btn`, `.container` styles reach all slot pages
